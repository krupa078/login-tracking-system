# Login Tracking System (Registration + Password + JWT + OTP + Rules)

This project contains:

- Backend (Node.js + Express + MongoDB)
- Frontend (React + create-react-app)
- Real email OTP support (Gmail/SMTP)
- JWT-based authentication
- Login history tracking with browser, OS, device, IP
- Conditional rules:
  - Chrome → OTP required
  - Microsoft Edge → no OTP
  - Mobile → only 10 AM–1 PM (IST)
